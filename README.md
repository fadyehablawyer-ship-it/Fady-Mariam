# 💍 موقع دعوة الزفاف الرقمية - Fady & Mariam

موقع ويب فاخر ومتجاوب (Responsive) كدعوة زفاف رقمية مبهرجة، جاهز للنشر على **GitHub Pages** برابط واحد فقط.

## ✨ المميزات
1. **ترحيب ديناميكي بالضيوف:** يتعرف على اسم الضيف تلقائياً عبر الرابط لإضفاء لمسة شخصية فاخرة.
2. **عداد تنازلي ذكي:** يحسب الأيام والساعات بدقة متناهية حتى موعد الإكليل.
3. **تكامل الخرائط والتقويم:** أزرار لفتح Google Maps وتحميل ملف `ICS` للتقويم بضغطة واحدة.
4. **تأكيد حضور ذكي عبر واتساب:** يرسل رسالة تلقائية مخصصة تحتوي على اسم الضيف المكتوب في الرابط.
5. **معرض صور أنيق:** يدعم عرض الصور بنظام الـ Lightbox المكبر ومؤثرات حركية ناعمة.

## 🚀 طريقة النشر على GitHub Pages

1. قم بإنشاء مستودع جديد (Repository) على حسابك في GitHub وسمّه مثلاً `wedding-invitation`.
2. ارفع ملف `index.html` والمجلد الخاص بالصور إلى المستودع.
3. اذهب إلى إعدادات المستودع (**Settings**).
4. من القائمة الجانبية، اختر **Pages**.
5. عند خيار **Build and deployment**، تأكد أن المصدر (Source) هو `Deploy from a branch`.
6. اختر فرعك الرئيسي `main` أو `master` ثم اضغط **Save**.
7. انتظر دقيقة واحدة، وسيظهر لك رابط الدعوة الخاص بك!

## 🔗 كيف تصنع روابط مخصصة للضيوف؟
بعد أن تحصل على رابط موقعك من GitHub Pages (مثال: `https://yourusername.github.io/wedding-invitation/`)، يمكنك إرسال روابط مخصصة لكل ضيف عبر إضافة `?guest=اسم_الضيف` في نهاية الرابط:

* لدعوة أحمد: `.../index.html?guest=Ahmed`
* لدعوة عائلة أستاذ محمد: `.../index.html?guest=الأستاذ محمد وعائلته الكريمة`

## 📸 تخصيص الصور والبيانات
* **الصور الافتراضية:** تم استخدام صور استعراضية من Unsplash. لتغييرها بصورك الخاصة، ضع صورك داخل مجلد `assets/images/` وقم بتعديل المسار (src) في قسم `` في ملف الـ `index.html`.
